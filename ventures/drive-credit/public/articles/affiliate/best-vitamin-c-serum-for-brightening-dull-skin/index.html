Does vitamin C serum help with acne scars?

It helps with discoloration from acne scars by reducing melanin production, but it won’t fix texture or depth. For dark spots from acne, La Roche-Posay Mela B3 is more targeted. For overall glow and prevention of post-acne dullness, vitamin C is ideal.

Why do some vitamin C serums turn yellow?

Yellowing means the vitamin C has oxidized and lost potency. La Roche-Posay uses airless packaging and stable formulations to prevent this. Avoid serums that come in clear bottles or don’t specify “stable” vitamin C.

Should I use vitamin C or niacinamide first?

Apply vitamin C first, then niacinamide. Vitamin C works best in a slightly acidic environment, so layer it before niacinamide, which has a higher pH. If you use both, wait 2–3 minutes between layers.
