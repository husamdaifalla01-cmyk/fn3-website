## POST-APPROVAL WORKFLOW

### When FlexOffers approves you:
1. Log in → go to "Find Advertisers" → search each target offer above
2. Apply to each individual advertiser program (separate approval per advertiser, usually 1–3 days)
3. Once approved for an offer, grab the tracking link from FlexOffers dashboard
4. Add to `affiliateUrls.ts` in the codebase under the appropriate slot

### When Impact approves you:
1. Log in → "Brands" tab → search each target advertiser
2. Submit partnership request with a note: *"Subprime credit audience, car owners, 300–600 score range"*
3. Most Impact brands review within 3–5 business days
4. Once approved, get your tracking link from the "Links" section of each partnership

### When CJ approves you:
1. Log in → "Advertisers" tab → search by category "Financial Services"
2. Apply to each advertiser with a one-line pitch about your audience
3. Add approved links to `affiliateUrls.ts`
