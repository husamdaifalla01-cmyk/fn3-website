## Before Posting Any Video

### Brand Compliance
- [ ] Brand name shown as "Mintbrooks" (not DriveCredit, not mintbrooks.com spelled out)
- [ ] Colors match brand: amber/gold, dark stone, emerald accent — no off-brand colors
- [ ] Font style: bold, clean — no decorative fonts

### Legal Compliance  
- [ ] "#ad" or "#sponsored" visible in on-screen text OR caption first line
- [ ] No "guaranteed" language
- [ ] No "no credit check" — only "soft inquiry" or "soft pull"
- [ ] "Up to" qualifier on any credit limit mention

### Content Quality
- [ ] Hook in first 3 seconds — is it compelling?
- [ ] Video is 15–30 seconds — not too short, not too long
- [ ] CTA at end is clear: "Link in bio" 
- [ ] Text is readable on mobile (not too small)
- [ ] Video does not end abruptly — proper fade/hold

### Technical
- [ ] Resolution: 1080x1920 (9:16 vertical)
- [ ] File size reasonable (<50MB for TikTok, <100MB for IG)
- [ ] Audio audible if present; captions present if no audio
