## The Funnel (Current State)

```
TikTok/IG → Bio Link → mintbrooks.com/links → mintbrooks.com → Yendo CTA → Lead
```

**Every step has a leak. We can't patch leaks without traffic. But we can pre-optimize.**

---

## Step-by-Step Leak Analysis

### Step 1: Video → Bio Link Click
- **Benchmark:** 3-5% CTR on financial TikToks
- **Our situation:** 0 videos posted → 0 data
- **Pre-optimization:** CTA in every video caption must say "Link in bio ↑" — not "check website"
- **Action:** All 4 posted captions include explicit "Link in bio 👆" call-to-action ✅

### Step 2: Bio Link → Site
- **Current:** mintbrooks.com/links page exists ✅
- **Risk:** Is the /links page copy compelling enough to push to site?
- **Recommendation:** Add social proof to /links: "4.8/5 from 1,200+ readers" (aspirational — update when real)
- **Action:** DEV task — add a social proof line to /links page hero

### Step 3: Site → Yendo CTA Click
- **Benchmark:** 8-15% CTR on affiliate finance sites
- **Our target:** 10% CTR minimum
- **Current:** Yendo CTA is on homepage hero, calculator, all guide pages
- **Weakness:** Homepage hero CTA is "Check If My Car Qualifies" — good. But secondary CTAs are less prominent.
- **Action:** Audit all 23 pages. Every guide page must have a Yendo CTA above the fold.

### Step 4: Yendo CTA → Lead
- **Benchmark:** 5-10% of Yendo clickers complete application (Yendo handles this)
- **Our job:** Send quality traffic (car owners, bad credit, right states)
- **Action:** Add state checker to /qualify before showing Yendo CTA
