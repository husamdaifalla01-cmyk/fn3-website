## Option A: Post Silent with Text Captions (FASTEST — do this NOW)

TikTok and IG Reels autoplay muted. Silent videos with strong text overlays perform WELL because most viewers watch without sound. The Remotion videos already have text overlays.

**Steps to post (TikTok):**
1. Transfer MP4 files to phone (AirDrop or Google Drive)
2. Open TikTok → + → Upload → Select video
3. Add trending sound at LOW VOLUME (10-15%) — helps algorithm discovery
   - Search "original sound" or use a trending credit/finance audio
4. Add captions from the copy-paste blocks below
5. Add hashtags from the copy-paste blocks below
6. **CRITICAL**: Set bio link to `mintbrooks.com/links` before first post
7. Post → repeat for all 4 videos (space 6-8 hours apart)

**Steps to cross-post (IG Reels):**
1. Same video → Instagram → Reels → Upload
2. Same caption (remove TikTok-specific hashtags, add IG ones)
3. **CRITICAL**: Set IG bio link to `mintbrooks.com/links`

## Option B: Add TTS Voiceover (BETTER — adds 5 min per video)

Use CapCut or TikTok's built-in text-to-speech:
1. Upload video to CapCut
2. Add text overlays (scripts below)
3. Select each text → "Text to Speech" → choose deep male voice
4. Adjust timing so speech matches animation transitions
5. Export → post to TikTok
