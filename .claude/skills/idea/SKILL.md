# /idea - Product Ideation & Validation

## Description
Rapid ideation and validation framework for new products, features, or business concepts. Combines CPO strategic thinking with CTO feasibility analysis to determine if an idea is worth pursuing.

## Commands

### `/idea [concept]`
Evaluate a product/feature idea against the FN3 5-test framework:

1. **Dent Test**: Does this make a dent in the universe?
2. **10x Test**: Is this 10x better than alternatives?
3. **Speed Test**: Can we ship MVP in one sprint?
4. **Moat Test**: Does this create defensible advantage?
5. **Scale Test**: Does this work at $50T portfolio scale?

### `/idea compare [idea1] vs [idea2]`
Compare two ideas head-to-head across all tests.

### `/idea backlog`
Review idea backlog with priorities and statuses.

---

## Ideation Principles

1. **Kill fast**: Bad ideas should die quickly
2. **Data over opinion**: Research beats intuition
3. **Unique advantage**: Only build what FN3 uniquely can
4. **User obsession**: Solve real problems, not imagined ones
5. **Compounding**: Prefer ideas that make other features better
